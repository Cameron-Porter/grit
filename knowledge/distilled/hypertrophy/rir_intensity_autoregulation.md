# Reps in Reserve (RIR) & Intensity Autoregulation

## Key Principles
* **Reps in Reserve (RIR):** A self-assessment tool estimating how many more repetitions could be performed with perfect form before reaching technical failure.
* **Technical Failure:** The threshold where a full-range-of-motion rep can no longer be completed with proper technique.
* **Autoregulation:** Adjusting training intensity based on daily performance capacity rather than static, rigid numbers.
* **Effective Volume:** Sets performed close enough to failure to trigger hypertrophy. Training too far from failure is ineffective; constant true failure induces excessive systemic fatigue.
* **Progression Mechanics:** Manage intensity over a mesocycle by starting with a higher RIR (3–4) and tapering toward 0–1 RIR as the cycle nears its end.

## Scientific Takeaways
* **Proximity to Failure:** Training close to failure is necessary for mechanical tension and muscle growth (Strong Evidence).
* **The "1-2" Sweet Spot:** There is no significant hypertrophy advantage to absolute failure versus stopping 1–2 reps short (Moderate Evidence).
* **Reliability:** RIR is a validated, reliable method for autoregulating intensity in trained populations (Moderate Evidence).

## Coaching Notes
* **RIR is a Skill:** Beginners require practice—and occasional testing—to calibrate their subjective sense of effort.
* **Technical Integrity:** If form breaks down, the RIR value is invalidated. Technique must remain the primary constraint.
* **Stimulus-to-Fatigue Ratio (SFR):** RIR is the best tool for balancing high-volume programs without inducing systemic burnout.
* **Consistency:** Maintain the same target RIR across all working sets in a session. Reps may drop as fatigue sets in, but the RIR target remains constant.

## Beginner Mistakes
* **Sandbagging:** Consistently underestimating effort (e.g., calling a set RIR 2 when it was actually RIR 8).
* **Over-adjusting:** Drastically changing weights based on daily mood rather than evaluating long-term trends across a mesocycle.
* **Sacrificing Form:** Grinding out "ugly" reps to hit a perceived failure point, adding fatigue without gaining effective stimulus.
* **Inconsistency:** Treating RIR differently across the first vs. the last set of a movement.

## Advanced Insights
* **Exercise Selection:** Use lower RIR (0–1) on stable isolation movements where form risk is low. Use higher RIR (1–2) on high-skill, injury-prone compound movements (e.g., squats, deadlifts).
* **Calibration:** Periodically perform safe, absolute failure sets to re-calibrate the "RIR gauge" against reality.

## App Implementation Ideas
* **Intensity Autoregulator:** 
    1. **Post-Set Logging:** User inputs RIR after every set.
    2. **Calibration Mode:** Periodic prompts for an AMRAP (As Many Reps As Possible) set to refine the user's perception of failure.
    3. **Fatigue Management:** Trigger a deload suggestion if the user reports RIR 0 frequently over several weeks.
    4. **Dynamic Progression:** Suggest weight increases if the user consistently hits lower-than-target RIR (e.g., hitting RIR 4 when targeting RIR 2).
