# G.R.I.T. Exercise Taxonomy

> Master classification of all exercises in the G.R.I.T. exercise library.
> This taxonomy powers the slot-based workout generator, exercise picker, and progression framework.
> Sources: knowledge/distilled/ (24 files), knowledge/docterine/exercise_selection.md

---

## How This Taxonomy Works

Every exercise belongs to exactly one **movement category** — a biomechanical grouping that defines the primary pattern, muscle demands, and fatigue profile. The slot-based workout generator uses movement categories to:

1. Assign exercises to the correct slot role (Primary / Secondary / Accessory)
2. Enforce session-level fatigue management (total high-fatigue movements per session)
3. Select appropriate rep ranges per slot
4. Ensure movement balance across a full program (push/pull/hinge/squat coverage)

## Movement Categories

### 1. Horizontal Press

**Movement pattern:** Pushing away from the torso in a horizontal or slightly declining plane.

| Field | Value |
|---|---|
| Primary muscles | Pectoralis Major (mid/lower), Triceps Brachii, Anterior Deltoid |
| Secondary muscles | Serratus Anterior, Coracobrachialis, Rotator Cuff |
| Fatigue rating | High (compounds) / Medium (flyes/cables) |
| Slot role | Primary (compounds) → Accessory (flyes/isolation) |
| Rep range — compound | 5–12 |
| Rep range — isolation | 10–20 |
| Beginner suitable | Yes |
| Progression suitability | All levels |
| Key cue | Full pec stretch at the bottom is the most productive range — do not shorten the eccentric |

### 2. Incline Press

**Movement pattern:** Pushing away from the torso at an upward angle (30–45°), emphasizing the clavicular (upper) head of the pectoralis major and the anterior deltoid.

| Field | Value |
|---|---|
| Primary muscles | Pectoralis Major (clavicular/upper), Anterior Deltoid, Triceps Brachii |
| Secondary muscles | Serratus Anterior |
| Fatigue rating | High (compound presses) / Medium (flyes) |
| Slot role | Secondary (when chest is Primary) → Primary (when upper chest is emphasized) |
| Rep range — compound | 6–15 |
| Rep range — isolation | 10–20 |
| Beginner suitable | Yes |
| Progression suitability | All levels |
| Key cue | Angle above 45° shifts load from upper chest to anterior delt — keep 30–45° |

### 6. Quad Dominant

**Movement pattern:** Knee-flexion/extension under load — squats, lunges, leg press. The primary knee extensor pattern.

| Field | Value |
|---|---|
| Primary muscles | Quadriceps (rectus femoris, vastus lateralis, medialis, intermedius) |
| Secondary muscles | Gluteus Maximus, Adductors, Erector Spinae (bilateral), Core |
| Fatigue rating | Very High (barbell squats) / High (machine squat, split squat) / Low (leg extension) |
| Slot role | Primary (barbell squat, leg press) → Secondary (split squat, lunge) → Accessory (leg extension) |
| Rep range — compound | 5–12 |
| Rep range — unilateral | 8–15 |
| Rep range — machine isolation | 12–25 |
| Beginner suitable | Yes (goblet squat, leg press); Barbell back squat requires form coaching |
| Progression suitability | All levels |
| Key cue | ROM drives quad development more than load — full depth is non-negotiable for maximum stimulus. For leg extensions: lean back slightly to place the hip in extension, stretching the rectus femoris (the only quad head that crosses both the hip and knee). Standard upright seating leaves the rectus femoris understretched — lean-back positioning is Strong Evidence for superior rectus femoris stimulus |

### 7. Hip Hinge

**Movement pattern:** Hip extension under load — deadlifts, RDLs, leg curls, good mornings. The primary hamstring/glute loading pattern that also involves the posterior chain.

| Field | Value |
|---|---|
| Primary muscles | Hamstrings (biceps femoris, semitendinosus, semimembranosus), Gluteus Maximus, Erector Spinae |
| Secondary muscles | Latissimus Dorsi (deadlift bracing), Trapezius, Adductors, Gastrocnemius |
| Fatigue rating | Very High (deadlift) / High (RDL, good morning) / Medium (leg curls) |
| Slot role | Primary (deadlift, RDL) → Secondary (leg curl variants) |
| Rep range — deadlift | 3–8 |
| Rep range — RDL/good morning | 6–12 |
| Rep range — leg curl | 10–20 |
| Beginner suitable | Leg curls: Yes; RDL: after basic coaching; Deadlift: requires form mastery |
| Progression suitability | All levels (leg curls); intermediate/advanced (deadlift) |
| Key cue | Hamstring tension at the stretched (hip-flexed) position drives stimulus — do not truncate the eccentric descent. Seated leg curl (★ Preferred) trains the hamstring at longer overall length than lying leg curl because the hip is flexed during the movement, adding the hip component of hamstring stretch to the knee component. This is rated Strong Evidence for superior hamstring hypertrophy |

## Fatigue Weighting for Session Design

The slot generator uses fatigue ratings to enforce the **SESSION_MAX_SETS = 24** and **SESSION_MAX_EXERCISES = 5** caps without overloading the session with high-fatigue movements.

| Fatigue Rating | Fatigue Points | Max per Session |
|---|---|---|
| very_high | 4 | 1 (deadlift or barbell squat only) |
| high | 3 | 2–3 |
| medium | 2 | 3–4 |
| low | 1 | Unlimited within set cap |

**Session fatigue cap:** 12 points total. A session with a deadlift (4) + two high-fatigue compounds (6) = 10 points — leaves room for 2 medium or lower accessories.

## Push / Pull / Hinge / Squat Balance

Every full program week should contain each of the four fundamental movement patterns to ensure structural balance and injury prevention:

| Pattern | Movement Category | Minimum per Week |
|---|---|---|
| Push (Horizontal) | Horizontal Press + Incline Press | 1 session |
| Push (Vertical) | Vertical Press | 1 session |
| Pull (Horizontal) | Horizontal Pull | 1 session |
| Pull (Vertical) | Vertical Pull | 1 session |
| Hinge | Hip Hinge | 1 session |
| Squat | Quad Dominant | 1 session |

A program lacking any of these patterns for more than 2 consecutive weeks has a structural imbalance that the generator must flag.
